# Конструктор архива тестов

Веб-приложение для подготовки архивов с тестами задач программирования. Интерфейс позволяет поштучно или пакетно вводить пары файлов с входными и выходными данными, управлять их порядком, редактировать и скачивать итоговый архив в формате ZIP.

## Возможности

- добавление тестов по одному и пакетная вставка из таблицы (например, из Google Docs);
- визуальный просмотр текущего набора тестов, редактирование, удаление и изменение порядка пар;
- формирование ZIP-архива с именами файлов `1`, `1.a`, `2`, `2.a` и т. д.;
- автоматическое формирование имени архива с учётом заданного префикса и текущих даты и времени;
- сохранение прогресса в `localStorage` и возможность начать новый архив одной кнопкой.

## Развёртывание

Страницы находятся в корне репозитория и готовы к публикации через GitHub Pages. Для локальной работы достаточно открыть `index.html` в браузере.
